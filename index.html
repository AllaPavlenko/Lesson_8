Pseudo class

  • Hover
  • Active
  • Focus
  • Target

The :hover CSS pseudo-class fires when the user hovers over an element with the mouse, but does not necessarily activate it. This style can be overridden by other link-related pseudo-classes such as :link, :visited, and :active appearing in the appropriate order.

The :active pseudo-class corresponds to an element when it is activated by the user. It allows the page to react when the element is activated. The interaction of an element with the mouse is typically the time between the user pressing and releasing the mouse button.

The :focus CSS pseudo-class is applied when an element (such as a form input) receives focus. It is usually activated when the user clicks the mouse or selects an element using the "tab" key on the keyboard.

The :target CSS pseudo-class represents a unique element (the target element) with a matching URL fragment id.
